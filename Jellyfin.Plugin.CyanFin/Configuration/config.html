🎬 CyanFin Plugin

Deep integration between Jellyfin and the CyanFin frontend. Provides trickplay thumbnails, watch parties, push notifications, and custom metadata.

URL of your CyanFin container. Leave empty to disable push notifications.
Optional secret to authenticate webhook calls between Jellyfin and CyanFin.

Trickplay

Generate scrubber preview thumbnails (requires ffmpeg).
How often to capture a thumbnail. 10s recommended.

Watch Party

Allow multiple CyanFin users to watch in sync.

Notifications

Notify CyanFin when items are added, playback starts/stops, etc.

Custom Metadata

Personal ratings, notes and tags per user, stored separately from Jellyfin metadata.
